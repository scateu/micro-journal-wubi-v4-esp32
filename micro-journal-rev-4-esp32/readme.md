INSTALL:

    cd micro-journal-rev-4-esp32
    pio run -e cardputer-adv -t upload


Additional Features:
 - Readline Keybindings: C-n C-p C-k C-y M-f M-b C-b C-d
 - External USB keyboard support
 - C-space switch IME
 - C-s Save
 - Fn-0..9 switch txt file
 - Ctrl -+ =_   increase decrease font size.
