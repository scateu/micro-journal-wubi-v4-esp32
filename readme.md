INSTALL

    cd micro-journal-rev-4-esp32
    pio run -e cardputer-adv -t upload


Additional Features:
 - C-space switch IME
 - C-s Save
 - Readline Keybindings: C-n C-p C-k C-y M-f M-b C-b C-d
 - External USB keyboard support: Logi USB, Thinkpad USB External Keyboard.. but not all, don't know why yet.
 - Ctrl -+ =_   increase decrease font size.
 - Fn-0..9 switch txt file (from upstream micro journal)
